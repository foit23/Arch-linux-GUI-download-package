# Arch-linux-GUI-download-package
    ru: Это программа с графическим интерфейсом для того чтобы вы могли устанавливать пакеты для Линукс дистрибутивов на базе Арч Линукс
-----------------------------------------------------------------------------------------------------------------------------------------------------
    en: This is a program with a graphical interface that allows you to conveniently install packages for Linux distributions based on Arch Linux.

**RU: КАК ИСПОЛЬЗОВАТЬ:**

**Способ установки 1:**  
1. Скачайте с GitHub ZIP-архив, содержащий исходный код программы.  
2. Распакуйте программу в удобном для вас месте.  
3. Откройте терминал и введите следующую команду:  
   --- `sudo python /abc/main.py` (вместо `abc` укажите путь к вашей папке и самому проекту, например: `~/Documents/Arch-linux-GUI-download-package/`) ---  
4. **!!!ВАЖНО!!!** Для работы программы у вас должен быть установлен **Python 3.14.3**. Проект, скорее всего, будет работать и на других версиях, но автор (я) использовал именно эту.  
5. Программа откроется. В поле ввода введите название пакета без ошибок. Найти его можно на сайте: [https://archlinux.org/packages](https://archlinux.org/packages).  
6. После завершения установки в терминале будет отображено название и версия пакета. В случае ошибок они также будут выведены там.

**P.S.** Это тестовая версия программы, она будет улучшаться и дорабатываться. Если вы хотите сообщить об ошибках, можете написать мне в Telegram: @foit_23.

**EN: HOW TO USE:**

**Installation Method 1:**  
1. Download the ZIP archive from GitHub containing the program's source code.  
2. Unzip the program in a convenient location.  
3. Open a terminal and enter the following command:  
   --- `sudo python /abc/main.py` (replace `abc` with the path to your folder and the project itself, e.g., `~/Documents/Arch-linux-GUI-download-package/`) ---  
4. **!!!IMPORTANT!!!** For the program to work, you must have **Python 3.14.3** installed. The project will likely work on other versions, but the author (me) used this one.  
5. The program will open. In the input field, enter the package name without errors. You can find it on the website: [https://archlinux.org/packages](https://archlinux.org/packages).  
6. Once the installation is complete, the terminal will display the package name and version. In case of errors, they will also be displayed there.
